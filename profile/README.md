# City RP Server

City RP is a roleplay server focused on immersive city-life experiences. Players can take on various roles such as law enforcement, emergency services, business owners, or everyday citizens in a dynamic and structured environment.  


## Websites & Resources

City RP also includes community management tools such as forums, a main website, and a Discord server to stay connected and organize events.  


## Getting Started

Players need a compatible game client. Once connected, they can create a character and start roleplaying according to server rules.  


## Support

Community support is available through Discord and forums for questions, bug reports, and assistance.
